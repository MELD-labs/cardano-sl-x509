Description

Configuration for generating X509 certificates

Users are expected to provide configuration (from file or by hand) and use fromConfiguration to turn them into description. Then, full description can be turned into certificates via genCertificate

Configuration --fromConfiguration--> Description --genCertificate--> Certificate
